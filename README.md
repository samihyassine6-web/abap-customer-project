# Sales Order ALV with SmartForms & Adobe Forms

## Overview

This project demonstrates an interactive ABAP application for displaying Sales Orders and Items using ALV.

It includes integration with:

* SmartForms
* Adobe Forms

---

## Features

* Interactive ALV Grid
* Display Sales Orders and Items
* Navigation between header and item data
* Custom buttons for form output

---

## Functionalities

### ALV Display

* Sales Order Header
* Sales Order Items
* Sorting, filtering, aggregation

---

### Interactive Actions

Custom buttons in ALV:

* Display SmartForm
* Display Adobe Form

---

### SmartForms Integration

* Classic SAP form output
* Structured document layout

---

### Adobe Forms Integration

* Advanced form layout
* PDF generation
* Modern print output

---

## Architecture

* ABAP Report Program
* Modularized logic
* Form handling classes

---

## Technologies

* ABAP
* ALV Grid
* SmartForms
* Adobe Forms

---

## Purpose

This project demonstrates real-world reporting scenarios with document output capabilities.

SAP NetWeaver AS ABAP 7.52 Developer Edition
